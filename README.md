## Run Locally

1. Clone the project

```bash
  git clone https://github.com/Chipl7/Matrix.git
```

2. Go to the project directory

```bash
  cd Matrix
```

3. Src dir

```bash
  cd src/cat
```

4. Run make for build

```bash
  make gcc
```

5. Run test

```bash
  make test
```
